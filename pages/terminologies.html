HL7 V2 Tables

HL7 V2 standard tables are published by HL7.org as ValueSet resources that can be utilised in profile definitions.
V2 Tables is Use
NameURIDescription
0203http://hl7.org/fhir/v2/0203Identifier Type

HL7 V2 tables are also commonly extended for Australian use. This has been often implied in AS4700.x series Australian Standards content and can be referenced in defined profiles. 'Shadow' V2 tables for AU extenstions can be defined for this content. Currently these are reference without a formal ValueSet defined. The intent is to implement these tables so can be used in validation.
